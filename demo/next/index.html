This is the beta version under development. Release notes:

  • Removed dependency from Google Charts (new built-in library)
  • Added Bar chart
  • Added new parameters for charst
    • chart-area : opacity settings for area charts
    • chart-line : line charts line width
    • chart-point: point width (line intersection)
  • Added realtime settings panel (click on the settings icon)

To start/stop a test for each gauge/chart click on the gauge/chart itself then click on the wrench icon on top right corner

To view the HTML for each gauge click on the title

Click on the settings icon to customize each element
